# Contributing Rules

## Scope
This repository hosts conditional axioms and open challenges.

## Submissions
All proof/counterexample attempts must:
- Use the issue template.
- Declare dependencies.
- Label results as:
  **Conditional (Assuming XYSTEM)** if applicable.

## Prohibited
- Unlabeled conditional results.
- New axioms without equivalence proof to XYSTEM.

## File placement
- Proof drafts → proofs/
- Counterexample constructions → counterexamples/

## Review policy
Maintainer will:
- Tag status (invalid / partial / accepted).
- Update COUNTEREXAMPLES.md or EQUIVALENCE.md if relevant.
